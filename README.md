### Word Cloud

Macro to generate word cloud in excel
Select range and run macro to generate word cloud. The selected range should contain headers as first row, keywords in first column and frequency in second column.

|**Keyword**|**Frequency**|
|---------------|:-----------|
|Smart|540|
|Active|350|
|Digital|345|
|Advanced|342|
|Innovative|320|
